This photo editor is written by Kieran Grayshon using C++ in Visual Studio, Xcode and using g++ for linux

KDG PhotoEditor has many features they are...

  • Image scaling
  • Image rotation
  • Image cropping
  • Image reflection
  • Layering
  • Adding text to an image
  • Exporting to jpg and png
